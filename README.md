# Areal HR Test – Week 2

## Описание

Реализация веб-приложения, в котором специалист по кадрам ведет учет сотрудников в нескольких организациях.

Используется:

MacBook macOS 15.6.1

- **Draw.io** — схема базы данных
- **Visual Studio Code** — IDE для разработки
- **NestJS** — бэкенд
- **PostgreSQL** — база данных локально
- **TypeORM** — ORM
- **joi** — валидация данных
- **Soft delete** — логическое удаление записей
- **Git / GitHub** — система контроля версий


## Основные команды Git

* `git add .` — добавить все изменения
* `git commit -m "message"` — зафиксировать изменения
* `git push origin main` — отправить изменения в удалённый репозиторий
* `git pull origin main` — получить изменения из удалённого репозитория
* `git log` — посмотреть историю коммитов
* `git status` — посмотреть статус файлов
* `git checkout <branch>` — переключиться на ветку
* `git branch <name>` — создать новую ветку
* `git merge <branch>` — слить ветки
